Description of Service

OPSBOT is a Discord bot designed for Roblox SEA Military division management. It provides the following functionality:

  • Automated SEA blacklist enforcement via Discord bans
  • Division EXP and member count tracking via Roblox and Google Sheets integration
  • Event hosting, logging, and attendance tracking
  • Server moderation tools including bans, kicks, mutes, and warnings
  • Server and message logging
  • Weekly division analytics reporting
  • Configurable module system for enabling and disabling features
